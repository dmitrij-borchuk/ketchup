# ketchup
Pomodoro timer with additional analytics features

### Run
```
yarn install
yarn run dev
```

### Test
```
yarn install
yarn run test
```

### Lint
```
yarn install
yarn run Lint
```

### Storybook
```
yarn install
yarn run storybook
```

### Release
```
yarn build
```
Commit you build
```
yarn version
git push -tags
```
Then you can add release notes here `https://github.com/dmitrij-borchuk/ketchup/releases`
